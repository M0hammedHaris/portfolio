# Product Overview

This is a personal portfolio website for Mohammed Haris, a Full Stack Software Engineer with 4+ years of experience specializing in backend systems and full stack applications.

## Key Features

- **Professional Portfolio**: Showcases experience, projects, skills, and education
- **Modern Design**: Clean, responsive interface with dark/light theme support
- **Interactive Sections**: Hero, About, Experience, Projects, Contact, and Footer
- **Performance Focused**: Built with Next.js for optimal loading and SEO
- **Accessibility**: Designed with proper semantic HTML and accessibility considerations

## Target Audience

- Potential employers and recruiters
- Clients seeking software development services
- Professional networking contacts
- Technical community members

## Core Sections

1. **Hero Section**: Introduction with profile photo and call-to-action
2. **About Section**: Professional bio and background
3. **Experience Section**: Work history and achievements
4. **Projects Section**: Portfolio of completed projects with tech stacks
5. **Contact Section**: Professional contact information and social links
6. **Footer**: Additional navigation and copyright information
