# Portfolio Project Guidance

This is a personal portfolio for Mohammed Haris built with Next.js 16, React 19, TypeScript, Tailwind CSS 4, shadcn-style components, Framer Motion, and next-themes.

Use the existing design system before adding new UI dependencies. Keep content centralized in `src/lib/data.ts`, preserve accessibility, and verify changes with `npm run lint` and `npm run build` when practical.

Prefer small, professional UI improvements over large visual rewrites. Maintain responsive behavior, light/dark theme support, semantic sections, and the established orange/slate visual direction.
